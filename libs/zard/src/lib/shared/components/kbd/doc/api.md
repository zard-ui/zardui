# API Reference

## z-kbd

Use `z-kbd` component to display a keyboard key.

| Input   | Type         | Default | Description            |
| ------- | ------------ | ------- | ---------------------- |
| `class` | `ClassValue` | `''`    | Additional CSS classes |

## z-kbd-group

Use `z-kbd-group` component to group `z-kbd` components together.

| Input   | Type         | Default | Description            |
| ------- | ------------ | ------- | ---------------------- |
| `class` | `ClassValue` | `''`    | Additional CSS classes |
