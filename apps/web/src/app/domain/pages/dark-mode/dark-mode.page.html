ZardUI implements a robust dark mode system using @custom-variant dark from TailwindCSS v4 combined with an Angular Appearance Service for state management and persistence. This service is part of core component and you can inject it where you need it.

Current Implementation

The current system works by applying the .dark class to the root HTML element, activating all dark style variations defined in the components.

Appearance Service

The core of the system is an Angular service that manages theme state, persistence, and CSS class application.

Usage in Application

The service is initialized in Zard provider. The HeaderComponent constains an example of how to use the service for toggle control.
