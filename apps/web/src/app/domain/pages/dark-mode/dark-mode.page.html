Implementing dark mode in ZardUI using TailwindCSS v4 custom variants and our DarkMode service.

ZardUI uses a robust dark mode system with @custom-variant dark and the DarkModeService for seamless theme switching.

TailwindCSS v4 Custom Variants

ZardUI leverages TailwindCSS v4's custom variants feature to create a powerful dark mode system that works with the class-based approach .dark.

Why Custom Variants?

The @custom-variant dark (&:is(.dark *)) approach provides better performance and more reliable dark mode detection compared to media queries, allowing users to manually toggle themes regardless of system preferences.

ZardUI DarkMode Service

Our DarkModeService provides the simplest and most reliable way to implement dark mode in any Angular project. This is the same service ZardUI uses internally.
