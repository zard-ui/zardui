How to use blocks

Learn how to add blocks to your project

1

Browse blocks

Browse the available blocks and find one that fits your needs. Click on a block to view its details.

2

Copy the code

Copy the component code and paste it into your project. All blocks are built with Angular and TailwindCSS.

3

Customize

Customize the block to fit your design. Change colors, spacing, and layout using TailwindCSS classes.

4

Ship it

That's it! Your block is ready to use. Build beautiful apps faster with pre-built components.
