Zard UI is built with modern Angular features like signal inputs, standalone components, and the new control flow syntax. We continuously update the library to leverage the latest capabilities while maintaining backward compatibility with recent Angular versions.

Support Policy

Zard UI supports the two latest major Angular versions . When a new major Angular version is released, we update Zard UI to support it and drop the oldest version from our support matrix.

Why two versions?

Supporting the two latest major versions gives teams enough time to plan and execute their Angular upgrades without losing access to Zard UI updates and new components. This approach balances stability with our ability to adopt new Angular features quickly.
