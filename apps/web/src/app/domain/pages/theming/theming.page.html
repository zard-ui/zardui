Convention

We use a simple background and foreground convention for colors. The background variable is used for the component background and the foreground variable is used for the text color.

TailwindCSS v4 + OKLCH

With TailwindCSS v4, all OKLCH configuration is now done directly in CSS using the @theme directive. This provides better color accuracy and no more JavaScript config files needed.

ZardUI Base Colors

Choose from one of the ZardUI base color themes. Each theme uses OKLCH color space and provides a complete set of color tokens that work together harmoniously in both light and dark modes.

Zinc (Default)

A modern, neutral color palette with subtle blue undertones using OKLCH color space.

Slate

A cool, professional color palette with blue-gray undertones in OKLCH format.

Stone

A warm, earthy color palette with brown undertones optimized in OKLCH.

Gray

A pure, neutral color palette without any color undertones in OKLCH format.

Neutral

Similar to Gray but with very subtle warm undertones using OKLCH color space.

💡

Choosing a ZardUI Base Color

Start with one of these OKLCH-based themes and customize from there. Each provides excellent contrast ratios and accessibility out of the box. You can always override specific colors later while keeping the foundation solid.
