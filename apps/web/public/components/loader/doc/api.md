# API Reference

[z-loader] Component

> `z-loader` is a component that provides a loading animation.

To customize the loader, pass the following props to the component.

| Property  | Description | Type                  | Default   |
| --------- | ----------- | --------------------- | --------- |
| `[zSize]` | Loader size | `default \| sm \| lg` | `default` |
