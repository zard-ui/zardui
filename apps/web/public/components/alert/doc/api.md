# API

## z-alert

| Property         | Description       | Type                          | Default   |
| ---------------- | ----------------- | ----------------------------- | --------- |
| `[zTitle]`       | Alert title       | `string \| TemplateRef<void>` | `-`       |
| `[zDescription]` | Alert description | `string \| TemplateRef<void>` | `-`       |
| `[zIcon]`        | Alert icon        | `string`                      | `-`       |
| `[zType]`        | Alert variant     | `default \| destructive`      | `default` |
