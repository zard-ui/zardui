# Zard UI - Product Overview

## Project Purpose
Zard is a modern Angular component library inspired by shadcn/ui, built with TailwindCSS and designed for developers who value beautiful, accessible, and completely free open source components. The project aims to provide a comprehensive set of UI components that are both aesthetically pleasing and functionally robust for Angular applications.

## Key Features & Capabilities

### Component Library
- **Modern Angular Components**: Built with Angular 21+ using latest features and best practices
- **TailwindCSS Integration**: Styled with utility-first CSS framework for consistent design
- **Accessibility First**: Components designed with accessibility standards in mind
- **TypeScript Support**: Full TypeScript support with proper type definitions
- **Standalone Components**: Leverages Angular's standalone component architecture

### Development Tools
- **CLI Tool**: Custom CLI for easy component installation and project setup
- **Component Registry**: Centralized registry system for component management
- **Documentation Website**: Comprehensive documentation with live examples
- **Installation Guides**: Automated generation of installation instructions

### Architecture
- **Monorepo Structure**: Nx-powered workspace with multiple projects
- **Component Library** (`libs/zard`): Core UI components
- **Blocks Library** (`libs/blocks`): Higher-level component compositions
- **CLI Package** (`packages/cli`): Command-line interface tool
- **Web Application** (`apps/web`): Documentation and showcase website

## Target Users & Use Cases

### Primary Users
- **Angular Developers**: Building modern web applications with Angular
- **UI/UX Developers**: Creating consistent and accessible user interfaces
- **Open Source Contributors**: Contributing to component library ecosystem

### Use Cases
- **Rapid Prototyping**: Quickly build Angular applications with pre-built components
